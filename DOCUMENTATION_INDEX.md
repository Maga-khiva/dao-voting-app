## 🎯 **Deployment Workflow Overview**

```
1. SETUP (5 min)
   ├─ Get Infura API key
   ├─ Create .env file
   └─ Get testnet ETH

2. DEPLOY (5-10 min)
   ├─ Run: npx hardhat run scripts/deploy.ts --network sepolia
   ├─ Save contract addresses
   └─ Update frontend config

3. BUILD (2 min)
   ├─ cd frontend && npm run build
   └─ Creates dist/ folder

4. DEPLOY FRONTEND (2-3 min)
   ├─ Push to GitHub
   ├─ Connect to Netlify
   └─ Get live URL

TOTAL TIME: ~20 minutes
COST: $0 (testnet)
```
